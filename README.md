# Hospital Management System

A Full Stack Hospital Management System developed using Spring Boot and React.

## Features

- Patient Management
- Doctor Management
- Appointment Scheduling
- Priority Queue Management
- Medical Records
- Prescriptions
- Billing System
- Revenue Dashboard
- Hospital Visit Tracking
- Dashboard Analytics
- PDF Report Generation
- Role Based Authentication

## Technology Stack

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL

### Frontend
- React
- Vite
- Recharts
- jsPDF
